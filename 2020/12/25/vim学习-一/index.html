基础保存插入操作

vim ：进入vim，默认为normal默示

normal模式下：

i ：插入模式

:wq ：保存并退出

[range]s/{string1}/{string2}/[flags]：替换，range为范围，%表示全部，string1为被替代的字符串，string2为替代的字符串，flags表示标志，如果是全局则用g。

例：%s/int/long long/u 表示用long long替代全部的int

返回键退出插入模式

ctr+z：强制退出normal

编译一个c++程序

vim x.cpp 创建一个cpp文件

g++ x.vpp 编译该文件

./a.out 运行

更改配置

cd /etc/vim

sudo vim vimrc

直接在后面编辑保存退出即可。
